    <h1 align="center">Trace - Secure File Sharing</h1>
</p>

Firebase is used for all the tasks of users such as user registration and login, file storing;
AES algorithm is used to encrypt and decrypt file of user.

## Features Integrated
- File Encrytion
- FIle Decrytion
- Cloud Storage
- Offline Storage
